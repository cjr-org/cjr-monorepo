# Team Ownership Map

This repo uses lightweight team ownership so folder boundaries line up with plausible platform teams.

## Teams

- `platform-engineering`: `apps/catalog-api`, `catalog/`
- `platform-experience`: `apps/catalog-web`, `apps/docs-site`
- `platform-operations`: `apps/admin-portal`, `docs/`
- `developer-platform`: `services/ingestion-worker`, `services/notification-service`, `packages/shared-config`
- `identity-platform`: `services/auth-service`
- `security-platform`: `services/audit-worker`
- `design-systems`: `packages/ui-kit`
